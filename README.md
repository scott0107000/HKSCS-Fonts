# 《香港增補字符集》外字免費開源解決方案
字言字型成功開發以HKSCS（香港增補字符集）外字編碼規格的字型，目的為在解決早期使用了HKSCS編碼之文件能在現今正確顯示字形，而不會亂碼。
## 字型
本字型解決方案包括３款風格：

- 字言HKSCS外字集-黑體（`Fancy-HKSCS-EUDC-Hei`）
- 字言HKSCS外字集-宋體（`Fancy-HKSCS-EUDC-Sun`）
- 字言HKSCS外字集-楷體（`Fancy-HKSCS-EUDC-Kai`）

每款風格皆包含HKSCS字符及Big5-ET字符，共5,120個字符。這些字符不另外映射至正字碼位，僅適合在HKSCS規格文件作為Fallback字型顯示，不推荐使用字型內編碼輸入字符。

## 鳴謝
思源黑體、全字庫正宋體、全字庫正楷體 提供字形。
